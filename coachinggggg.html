Ja, "dachte" und "wollte" sind Verben, genauer gesagt finite Verbformen.

  • "dachte" ist die Präteritum-Form des Verbs "denken".
  • "wollte" ist die Präteritum-Form des Modalverbs "wollen".

Modalverben wie "wollen" stehen oft in Verbindung mit einem weiteren Verb im Infinitiv (z. B. "Ich wollte schlafen"), während Vollverben wie "denken" eigenständig eine Bedeutung tragen (z. B. "Ich dachte über das Problem nach").

Schritt 2: Neues Repository erstellen

  1. Melde dich bei GitHub an.
  2. Klicke oben rechts auf das “+” Symbol → “New repository”.
  3. Gib einen Namen ein, z. B. fitness-coaching.
  4. Setze das Repository auf Public (damit GitHub Pages funktioniert).
  5. WICHTIG: Aktiviere die Option "Add a README file".

  1. Klicke in deinem neuen Repository auf "Add file" → "Upload files".
  2. Ziehe deine HTML- und CSS-Dateien (die du von mir bekommen hast) in das Fenster.
  3. Klicke auf "Commit changes".

Schritt 4: GitHub Pages aktivieren

  1. Gehe zu den Settings deines Repositorys.
  2. Scrolle runter zu "Pages".
  3. Unter "Branch" wähle "main" aus und klicke auf "Save".
  4. Nach ein paar Minuten wird dir die URL deiner Website angezeigt (meistens https://dein-github-name.github.io/fitness-coaching/).

Schritt 5: Website testen

  • Besuche deine neue URL in einem Browser.
